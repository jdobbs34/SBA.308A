# Recipe Finder Web Application

## Project Description
Recipe Finder is a single-page web application that allows users to search, browse, and discover recipes from around the world. The app fetches data from TheMealDB API and provides an engaging, responsive user interface for exploring new meals.

## Features
- **Recipe Search**: Search for recipes by name with real-time results
- **Category Filtering**: Browse recipes by category (Seafood, Vegetarian, Dessert, Beef, Chicken, Pasta)
- **Paginated Gallery**: Navigate through search results with pagination controls
- **Recipe Details**: View full recipe details including ingredients and instructions in a modal
- **Random Recipe**: Discover new recipes with the random recipe feature
- **Responsive Design**: Mobile-friendly interface that works on all devices
- **Featured Recipes**: Random recipes displayed on initial page load
